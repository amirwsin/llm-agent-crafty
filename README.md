## هدف پروژه

این ریپو **فقط برای یک پروژه دانشگاهی در درس LLM** ایجاد شده است.

هدف‌های اصلی این پروژه عبارت‌اند از:

* آشنایی با نحوه کار **مدل‌های عامل‌مبنا (ABM)** در مسائل مربوط به سیاست‌گذاری
* بررسی این‌که **مدل‌های زبانی بزرگ (LLM)** چگونه می‌توانند به‌عنوان عامل تصمیم‌گیر یا نهاد سیاست‌گذار عمل کنند
* درک ساختار عامل‌ها، نحوه تصمیم‌گیری آن‌ها و تعاملشان با یکدیگر
* آزمایش و بهبود ایده‌های مربوط به عامل‌های سیاست‌گذار مبتنی بر LLM
* شناخت محدودیت‌ها و چالش‌های استفاده از LLM در شبیه‌سازی سیاست‌ها
